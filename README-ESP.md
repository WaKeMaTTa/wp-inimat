WP Inimat
=========

Es un plugin para wordpress llamado wp-inimat, creado para el proyecto Inimat. Compatible con *WordPress 3.5*

Funciones que proporciona el plugin:
* Añadir criaturas [en proceso]
* Modificar criaturas (opción solo para administradores)  [proximamente]
* Ver todas las criaturas  [proximamente]
* Ver un sola criatura con sus características  [proximamente]

## Instalación

Para instalar el plugin siga estos sencillos pasos:
* Descargue el archivo plug-in
* Extraer el archivo en el directorio `wp-content/plugins`
* Activar el plugin a través del menú de Plugins en WordPress
* Comenzar a usar los códigos cortos como se describió anteriormente

## Configuración

En estos momentos no tiene ningun panel para administrar-lo.
